
```css
.example::before {
  content: "★ ";
}
```

Inserts generated content **before** the element’s content.

---

```css
.example::after {
  content: " ✦";
}
```

Inserts generated content **after** the element’s content.

---

These pseudo-elements are often used for decoration, icons, quotes, or helper styling without extra HTML.
