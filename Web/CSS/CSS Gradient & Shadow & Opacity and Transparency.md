
```css
.example1 {
  background: linear-gradient(to right, red, blue);
}
```

Linear gradient — smooth transition from red to blue along a straight horizontal line.

---

```css
.example2 {
  background: radial-gradient(circle, yellow, green);
}
```

Radial gradient — colors radiate outward from the center in a circular pattern.

---

```css
.example3 {
  background: conic-gradient(from 0deg, pink, purple, cyan);
}
```

Conic gradient — colors transition around a central point like pie slices.

---

```css
.example4 {
  background: repeating-linear-gradient(45deg, orange, orange 20px, white 20px, white 40px);
}
```

Repeating gradient — linear gradient that repeats every 40px at a 45° angle.

---

```css
.example1 {
  box-shadow: 4px 4px 10px rgba(0, 0, 0, 0.3);
}
```

Box shadow — adds a soft shadow around the element’s box.

---

```css
.example2 {
  text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);
}
```

Text shadow — casts a shadow behind the text for depth or glow effects.

---

```css
.example3 {
  filter: drop-shadow(6px 6px 8px rgba(0, 0, 0, 0.6));
}
```

Drop shadow filter — applies a shadow to the rendered shape (works well with images, including transparent ones).
