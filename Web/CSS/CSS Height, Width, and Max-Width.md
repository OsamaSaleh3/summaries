
```css
.example {
  width: 300px;
}
```

Sets element’s width to a fixed 300 pixels.

---

```css
.example {
  max-width: 800px;
}
```

Sets maximum width to 800 pixels, preventing it from growing wider.

---

```css
.example {
  height: auto;
}
```

Height adjusts automatically based on content or aspect ratio.

---

```css
.example {
  height: 200px;
}
```

Sets element’s height to a fixed 200 pixels.
