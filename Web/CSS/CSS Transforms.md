```css
.example1 {
  transform: translate(20px, 20px);
}
```

Moves the element 20px to the right and 20px down.

---

```css
.example2 {
  transform: rotate(45deg);
}
```

Rotates the element 45 degrees clockwise.

---

```css
.example3 {
  transform: scale(1.5, 0.75);
}
```

Scales the element to 150% width and 75% height.

---

```css
.example4 {
  transform: skew(20deg, 10deg);
}
```

Skews (shears) the element 20° horizontally and 10° vertically.

---

```css
.example5 {
  transform: rotate(-15deg) translateX(30px) scale(1.2);
}
```

Applies multiple transforms: rotates 15° counterclockwise, moves 30px right, and scales up by 20%.
